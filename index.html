Favorite Food: Biryani

I love Biryani because of its rich flavors, aromatic spices, and delicious taste. It is a perfect combination of rice, meat, and spices, making it a fulfilling and flavorful meal.

How to Prepare Biryani

  1. Prepare Ingredients
    1. Wash and soak the rice.
    2. Chop vegetables or marinate the meat.
  2. Cook the Biryani
    1. Cook the rice separately.
    2. Prepare the masala with onions, tomatoes, and spices.
    3. Layer the rice and masala and let it simmer.
  3. Serve hot with raita or curry.

Side Dishes for Biryani

  • Raita
    • Yogurt
    • Cucumber
  • Mirchi Ka Salan
  • Boiled Egg

    import threading
import time
from queue import Queue


def f(n):
    time.sleep(n)


class Worker(threading.Thread):
    def __init__(self, queue):
        super(Worker, self).__init__()
        self.q = queue
        self.daemon = True
        self.start()

    def run(self):
        while 1:
            f, args, kwargs = self.q.get()
            try:
                f(*args, **kwargs)
            except Exception as e:
                print(e)
            self.q.task_done()


class ThreadPool(object):
    def __init__(self, thread_num=10):
        self.q = Queue(thread_num)
        for i in range(thread_num):
            Worker(self.q)

    def add_task(self, f, *args, **kwargs):
        self.q.put((f, args, kwargs))

    def wait_complete(self):
        self.q.join()
